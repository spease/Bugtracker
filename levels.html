Class descriptions

Class A:
Class A bugs crash the program, crash the computer, or force the user to stop using the application and terminate it.
Class B:
Class B bugs restrict gameplay or the usage of the program to the point where it's not longer fun or worth running.
Class C:
Class C bugs are primarily cosmetic. Misspelled text, bad graphics, textures not aligned properly, etc.
Class D:
Class D bugs are essentially feature requests.
